# pebble-kdeconnect
kdeconnect interface for pebble supported by termux to bridge to between the pebble and kdeconnect apps.

you need to install the pebble app in the repebble appstore, install termux for android, and then from termux run:

curl -s https://raw.githubusercontent.com/JacobNussbaum/pebble-kdeconnect/main/setup.sh | bash
